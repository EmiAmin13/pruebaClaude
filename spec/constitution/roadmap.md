# Roadmap

## Hecho ✅

1. **000 · Sistema de diseño** — CSS variables, navbar, sidebar, footer, cards, layout global, modo claro/oscuro. Contraste WCAG y AXE verificados automáticamente; responsive validado a mano.
2. **001 · Landing page** — Hero como `spec.md` renderizado, con los módulos del curso como criterios de aceptación.
3. **002 · Fundamentos de IA** — LLMs, tokens, ventana de contexto, alucinaciones, anatomía del prompt y vibe coding vs ingeniería.
4. **003 · OpenCode** — Instalación, comandos, Build vs Plan, AGENTS.md. Comandos verificados contra la documentación oficial vía Context7.
5. **004 · MCP y Skills** — Qué es MCP, en qué se diferencia de una skill, Context7 y cómo instalar skills.
6. **005 · Ciclo SDD** — El problema que resuelve, el ciclo, los tres niveles y los artefactos.
7. **006 · Multiagentes y loops** — Subagentes, flujo coordinador/implementador/verificador y loop engineering.
8. **007 · Recursos** — Material del curso, documentación oficial y herramientas.
