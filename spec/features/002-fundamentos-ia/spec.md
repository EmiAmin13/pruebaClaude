# 002 · Fundamentos de IA

**Estado:** propuesta

## Qué hace

Página que presenta los fundamentos de inteligencia artificial y LLMs vistos en el Día 1 del curso: cómo funcionan los modelos de lenguaje, tokens, parámetros, ventana de contexto, prompt engineering, RAG, y la diferencia entre vibe coding e ingeniería de software.

## Por qué

Es la base conceptual necesaria para entender el resto del curso. Sin estos conceptos, el uso de agentes de IA carece de contexto técnico.

## Criterios de aceptación

- [ ] Explicación visual de cómo funciona un LLM (diagrama o esquema).
- [ ] Secciones: LLMs, tokens, parámetros, contexto, multimodalidad, alucinaciones.
- [ ] Anatomía del prompt perfecto (rol, contexto, tarea, restricciones, formato).
- [ ] Comparativa vibe coding vs ingeniería de software.
- [ ] Diseño responsive.
- [ ] Pasa AXE checks.

## Fuera de alcance

- Fine-tuning en profundidad (solo mención conceptual).
- RAG implementado (solo explicación teórica).
