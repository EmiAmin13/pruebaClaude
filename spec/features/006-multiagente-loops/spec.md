# 006 · Multiagentes y loops

**Estado:** propuesta

## Qué hace

Página sobre patrones multiagente y loop engineering. Explica el uso de subagentes para tareas especializadas, el flujo coordinador → implementador → verificador, y cómo diseñar bucles autónomos de retroalimentación.

## Por qué

Completa la formación del Día 3 del curso. Los patrones multiagente son clave para escalar el desarrollo con IA más allá de una sola sesión.

## Criterios de aceptación

- [ ] Explicación de qué es un subagente y por qué usarlo.
- [ ] Diagrama del flujo multiagente (coordinador, implementador, verificador).
- [ ] Explicación de loop engineering y el bucle de verificación.
- [ ] Relación con SDD y OpenCode.
- [ ] Diseño responsive.
- [ ] Pasa AXE checks.

## Fuera de alcance

- Implementación de un sistema multiagente funcional.
- Comparativa con otras herramientas de orquestación.
