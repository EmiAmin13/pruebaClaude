## Qué hace

Página de recursos adicionales: guía esencial de Claude Code (PDF del curso), plantilla SDD descargable, enlaces a herramientas y documentación oficial.

Centraliza todo el material complementario en un solo lugar para que los alumnos lo tengan siempre accesible.

- [ ] Enlaces a herramientas: OpenCode, Claude Code, Cursor.
