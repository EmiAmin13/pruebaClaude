## Qué hace

Página sobre OpenCode: el agente de código abierto AI-first. Cubre instalación, configuración, comandos principales, modos Build vs Plan, y cómo crear y usar AGENTS.md.

OpenCode es la herramienta principal del curso. Los alumnos necesitan una referencia rápida de instalación, comandos y configuración.

- [ ] Diseño responsive.
