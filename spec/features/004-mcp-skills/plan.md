## Enfoque

Página con dos mitades claras: MCP y Skills. Incluir ejemplos de configuración (bloques de código JSON) y comandos de terminal.

## Implementación

1. Generar componente `McpSkillsPageComponent` en `src/app/pages/mcp-skills/`.
2. Configurar ruta lazy.
3. Maquetar sección MCP: qué es, cómo funciona, ejemplo con Context7.
4. Mostrar configuración JSON de Context7 en OpenCode.
5. Maquetar sección Skills: qué son, diferencia con MCP, instalación.
6. Incluir comandos `npx skills add` con ejemplos reales.

## Decisiones

- **Dos secciones independientes** — Fáciles de navegar y referenciar.
- **Bloques de código JSON** — Para la configuración MCP.
- **Comandos de terminal** — Con formato code block.
