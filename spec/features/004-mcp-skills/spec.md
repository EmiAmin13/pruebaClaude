# 004 · MCP y Skills

**Estado:** propuesta

## Qué hace

Página sobre el Model Context Protocol (MCP) y las Skills en OpenCode. Explica qué son los MCP servers, cómo configurar Context7, y cómo instalar skills como frontend-design y find-skills.

## Por qué

MCP y Skills son dos de las herramientas más potentes de OpenCode. Los alumnos necesitan entender la diferencia, cómo configurarlas y ejemplos prácticos.

## Criterios de aceptación

- [ ] Explicación de qué es MCP y para qué sirve.
- [ ] Diferencia entre MCP (servicios externos) y Skills (conocimiento interno).
- [ ] Tutorial de instalación de Context7 como MCP.
- [ ] Tutorial de instalación de skills (frontend-design, find-skills).
- [ ] Diseño responsive.
- [ ] Pasa AXE checks.

## Fuera de alcance

- Listado exhaustivo de todos los MCP disponibles.
- Creación de skills personalizadas.
